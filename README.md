#  VIX Prediction & Market Risk Dashboard Design

This repository contains an interactive R Shiny dashboard and HTML Markdown which analyzes and predicts stock market volatility (VIX). The app features real-time economic data to provide quantiative statistics and insights into financial market trends.

##  Features
-  **Data Pipeline** Fetches and processes most recent FRED API data for downstream analysis
-  **Predictive Modeling** Implements various financial models to estimate stock volatility
-  **Interactive Visuals** Produces dynamic charts using ggplot2. gganimate2, and plotly
-  **Risk Assessment** Includes predictive intervals and summary statistics for model fit

##  Technical Specifications
-  **Language** R
-  **APP** Shiny, bslib
-  **Packages** `tidyverse`, `fredr`, `vars`, `MASS`, `plotly`, `gganimate`
-  **Deployment** shinyapps.io
